# ATM-GUI

Aplikasi ini adalah aplikasi java sederhana tentang ATM. Pada aplikasi ini, user dapat melakukan :
1. Pengecekan Saldo
2. Penarikan Tunai
3. Transfer
4. Setor Tunai
